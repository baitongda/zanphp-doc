5.5. trace.php¶

5.5.1. 文件位置¶

resource/config/$ENV/monitor/trace.php

5.5.2. 配置作用¶

调用链相关配置

5.5.3. 配置文件内容¶

<?php

return [
    //是否开启调用链上报
    //如果开启需要先配置trace所需的tcp连接池
    "run" => false,
];

如果开启调用链，则需要在resource/config/$ENV/connection下配置tcp.trace连接池，具体参考Tcp连接池配置。
